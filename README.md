## NeoClassic Balls — визуализация логики шариков

Проект состоит из двух основных модулей:
- **`logic.py`** — чистая игровая логика (движение шариков, смешивание цветов, инвентарь, зона удаления).
- **`start.py`** — визуальный интерфейс на `pygame` в неоклассическом стиле, использующий `GameLogic`.

### Установка

- **Требования**: Python 3.9+ (рекомендуется), интернет для установки зависимостей.

1. Клонируй репозиторий или скачай проект.
2. Установи зависимости (главная — `pygame`):

```bash
pip install pygame
```

Если используешь `py` на Windows:

```bash
py -m pip install pygame
```

3. Убедись, что IDE (Cursor / VS Code) использует тот же Python-интерпретатор, где установлен `pygame`.

### Запуск

Из корня проекта:

```bash
python start.py
```

или

```bash
py start.py
```

Откроется окно с неоклассическим интерфейсом, в котором отображается мир шариков и панель инвентаря.

### Управление

- **SPACE** — создать новый шарик в центре сцены.
- **ПКМ (правая кнопка мыши, держать)** — всасывать шарики вокруг курсора в инвентарь.
- **ЛКМ (левая кнопка мыши, клик)** — выплюнуть один шарик из инвентаря в точку под курсором.
- **ESC** или закрытие окна — выход из программы.

Внизу игрового поля есть прямоугольная **зона удаления**: если шарик попадает в неё, логика (`GameLogic._remove_balls_in_delete_zone`) окончательно удаляет его из мира.

### Структура проекта

- **`logic.py`**
  - `Ball` — модель шарика.
  - `Inventory` — логический инвентарь.
  - `DeleteZone` — прямоугольная зона удаления (`x1, y1, x2, y2`).
  - `GameLogic` — основной класс симуляции: `create_ball`, `update`, `suck_balls_with_mouse`, `spit_ball_from_inventory`, списки `balls` и `inventory.balls`.
- **`start.py`**
  - Создаёт окно `pygame`, настраивает неоклассический UI.
  - Инициализирует `GameLogic` с размерами мира и `DeleteZone`.
  - В основном цикле каждый кадр:
    - обрабатывает события клавиатуры и мыши;
    - вызывает `logic.update(dt)`;
    - отрисовывает мир, шарики, зону удаления и панель инвентаря.

### Как доработать

- **Изменить визуальный стиль** — правь палитру в классе `Palette` в `start.py`.
- **Поменять размеры мира / окна** — правь константы `WINDOW_WIDTH`, `WINDOW_HEIGHT` и/или геометрию `world_rect` в `GameUI.__init__`.
- **Усложнить механику** — добавляй логику в `GameLogic` или новые методы во `logic.py`, при необходимости вызывая их из `start.py`.
